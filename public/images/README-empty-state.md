# Imagens do empty state (Figma)

Os empty states do painel usam as imagens abaixo. Para usar o design do Figma:

1. Abra o arquivo [SoftFlow no Figma](https://www.figma.com/design/kqpFqgT0lp8SgCTejjlUiP/SoftFlow?node-id=116-2314).
2. Exporte as ilustrações do empty state (node 116-2314 ou frames correspondentes).
3. Substitua os arquivos mantendo os mesmos nomes:

| Arquivo | Uso |
|--------|-----|
| `empty-state-produtos-priorizados.svg` | Card "Produtos Priorizados" |
| `empty-state-retorno.svg` | Card "Retorno" |
| `empty-state-casos-produto.svg` | Card "Casos do Produto" (sem seleção e sem casos) |

Formatos aceitos: SVG ou PNG. Se usar PNG, atualize a extensão nos componentes em `components/painel/*.tsx`.
